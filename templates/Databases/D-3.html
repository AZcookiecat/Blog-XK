mysql报错之 ERROR 1064 (42000)

作者：Ciin7mo 发布时间：2025.03.21

问题描述：You have an error in your SQL syntax; check the manual that corresponds to your MySQL server version for the right syntax to use near 'WHERE `id` = %s' at line 1


解决方案：

解决办法：（语法错误）检查sql语句是否正确，检查字段是否存在，检查字段类型是否正确。


(1)用ALTER USER 'root'@'localhost' IDENTIFIED BY '新密码'; 然后flush privileges; 刷新权限。


(2)或者UPDATE mysql.user SET Password=PASSWORD('新密码') WHERE User='root' AND Host='localhost';
然后FLUSH PRIVILEGES;


(3)或者直接采用 set password '你的密码'； 即可修改
然后再flush privileges; 刷新权限。
