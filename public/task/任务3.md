# 智慧中医构建任务详情

## 任务列表
1. 环境搭建：
- [ ] 阅读项目md文件，了解项目基本情况。
- [ ] 所有成员需下载安装配置好Mysql。Navicat可视化工具可装可不装。
- [ ] 所有成员拉取代码并在本地构建一个仓库，编写好的代码提交到master分支，经代码审核之后会合并到main分支

2. 前端：
- [ ] 完成页面设计，包括登录页面，注册页面，个人中心页面，中医知识展示页面等；除登录注册这些主要功能页外，展示页面不少于5个。

3. 后端：
- [ ] 学习Flask框架，完成后端路由设计，完成基础的页面跳转功能。
